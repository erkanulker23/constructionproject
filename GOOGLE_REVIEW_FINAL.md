# ✅ Google Yorumları Modülü - HAZIR!

## 🎯 Sistem Nasıl Çalışıyor?

### Çok Basit - 3 Adım:

#### 1️⃣ Google Maps Linkini Alın
```
Google Maps'te işletmenizi bulun
→ Paylaş butonuna tıklayın
→ Linki kopyalayın
```

**Örnek Link:**
```
https://www.google.com/maps/place/Rota+Nakliyat+%26+Depolama/@40.9562253,29.1506936...
```

#### 2️⃣ Admin Panele Yapıştırın
```
Admin Panel → İçerik Yönetimi → Google Yorumları
→ "Google Maps URL" alanına yapıştırın
→ [Kaydet ve Yorumları Çek] butonuna tıklayın
```

**Otomatik Olur:**
- ✅ URL'den işletme adı çıkarılır: "Rota Nakliyat & Depolama"
- ✅ İşletme bilgileri kaydedilir
- ✅ Sistem hazır!

#### 3️⃣ Yorumları Ekleyin
```
Aynı sayfada, aşağıdaki tabloda:
→ Yorumlarınızı manuel ekleyebilirsiniz
→ Yayınla/Yayından Kaldır
→ Sil
```

## 🏠 Homepage Builder'a Ekleme

```
Admin Panel → Anasayfa → Anasayfa İçeriği
→ + Blok Ekle
→ Google Yorumları seç
→ Ayarları yap:
   • Başlık: "Müşterilerimiz Ne Diyor?"
   • Alt Başlık: "Değerli görüşler"
   • Görünüm: Variant 1, 2, 3 veya 4
   • Yorum Sayısı: 6-10
   • Minimum Puan: 4 veya 5 yıldız
   • Arkaplan: Resim veya renk
→ Kaydet
```

---
